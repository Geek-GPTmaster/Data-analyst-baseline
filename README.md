# Data-analyst-baseline
A work flow shows the whole day of a business analyst.

Process

1️⃣ 数据获取｜Data Collection
中文笔记
• 使用 API 获取数据
• 使用 Web Scraping 技术抓取网页数据
English Notes
• Collecting data using APIs
• Extracting data through web scraping techniques

2️⃣ 数据清洗与整理｜Data Wrangling
中文笔记
• 处理重复数据（duplicates）
• 处理缺失值（missing values）
• 整理和规范数据结构
English Notes
• Handling duplicates
• Dealing with missing values
• Cleaning and structuring datasets

3️⃣ 数据分析｜Data Analysis
中文笔记
• 分析数据分布（data distributions）
• 处理异常值（outliers）
• 识别变量之间的相关性（correlations）
English Notes
• Analyzing data distributions
• Handling outliers
• Identifying correlations between variables

4️⃣ 数据可视化｜Data Visualization
中文笔记
• 根据数据类型选择合适的图表
• 使用图表、折线图、直方图等
• 用可视化清晰表达趋势和洞察
English Notes
• Selecting appropriate visualizations
• Using charts, plots, and histograms
• Communicating insights and trends visually

5️⃣ 仪表盘设计｜Dashboard Building
中文笔记
• 设计仪表盘
• 学习仪表盘布局与信息组织原则
• 让数据“可看、可用、可决策”
English Notes
• Designing dashboards
• Applying dashboard-building best practices
• Structuring data for effective decision-making

成果输出｜Reporting & Presentation
中文笔记
课程最后强调：
• 如何总结分析结果
• 报告结构设计
• 数据故事（data storytelling）
• 演示与展示的最佳实践
English Notes
• Summarizing analytical findings
• Structuring effective reports
• Presentation best practices
